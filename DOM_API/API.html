DOM API

This is the (Document Object Model ) API in Javascript.
It provides a way to interact with and manipulate HTML and XML documents.
It represents the structure of a document as a tree of nodes, where each node corresponds to a part of the document (like elements, attributes, and text) Using the DOM API, you can modify the structure, style, and content of a webpage dynamically.

  1. Document Object

    The document object is the entry point to the DOM. It represents the entire web page and allows access to its elements.
    // Example: Accessing the document title console.log(document.title); // Logs the title of the page

  2. Selecting Elements:

    You can select elements from the DOM using various methods:

    • getElementById(): Selects an element by its id.
    • getElementsByClassName(): Selects elements by their class name.
    • getElementsByTagName(): Selects elements by their tag name.
    • querySelector(): Selects the first element that matches a CSS selector.
    • querySelectorAll(): Selects all elements that match a CSS selector.
    // Example: Selecting an element by its ID const header = document.getElementById('header'); // Example: Selecting all elements with a class name const items = document.getElementsByClassName('item'); // Example: Selecting an element using a CSS selector const firstItem = document.querySelector('.item');

  3. Manipulating Elements:

    You can modify the content, attributes, and styles of elements.

    • Content: You can change the content of elements using innerHTML, textContent, and other properties.
    • Attributes: Methods like setAttribute(), getAttribute(), removeAttribute() are used to manipulate element attributes.
    • Styles: You can modify inline styles using the style property
    // Example: Changing the content of an element header.innerHTML = "New Title"; // Example: Changing an attribute firstItem.setAttribute('data-value', '123'); // Example: Modifying styles firstItem.style.color = 'red';

  4. Creating and Appending Elements:

    You can create new elements and insert them into the DOM using methods like createElement() and appendChild(). // Example: Creating and appending a new element const newItem = document.createElement('li'); newItem.textContent = 'New List Item'; document.querySelector('ul').appendChild(newItem);

  5. Event Handling:

    You can attach event listeners to elements to respond to user interactions, like clicks, keypresses, etc. The most common method is addEventListener(). // Example: Adding an event listener firstItem.addEventListener('click', () => { alert('Item clicked!'); });

  6. Traversal and Manipulation:

    The DOM API provides ways to traverse through the document tree, allowing you to move between parent, child, sibling nodes.

    • parentNode, childNodes, firstChild, lastChild, nextSibling, and previousSibling are some properties for traversal.
      • // Example: Traversing to the parent node console.log(firstItem.parentNode); // Example: Traversing to the next sibling console.log(firstItem.nextSibling);
